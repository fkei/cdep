# cdep
Classic Go dependency tool

# install

```
$ sh -c "$(curl -fsSL https://raw.githubusercontent.com/fkei/cdep/develop/install)"
```

> Environment: `export CDEP_PATH={{INSTALL PATH}}`

> append .gitignore -> ./vendor

# Use

Install and Update

```
$ ./vendor install
```

Uninstall

```
$ ./vendor uninstall
```

Help

```
$ ./vendor help
```
